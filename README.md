# Solar Power System Calculator

## Overview
This project is a Solar Power System Calculator designed to help users estimate the power output of a solar panel system based on various parameters such as location, panel specifications, and weather conditions.

## Features
- Calculate estimated solar power generation based on user inputs.
- User-friendly interface for inputting system details.
- Provides output in kWh and other relevant metrics.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/chealseainfo1/PV-array.git
   ```
2. Navigate to the project directory:
   ```bash
   cd PV-array
   ```
3. Install dependencies (if applicable).

## Usage
- Input the required parameters: panel wattage, average sunlight hours, and efficiency.
- Click the "Calculate" button to get the estimated power output.
